Hi there! My name is Leah and I'm a full-stack developer.

My Projects:

  • Hangman

    GitHub | Application

    About this project

    Details: This is a hangman game that I
    created with HTML, CSS and
    JavaScript. The user has to guess the
    letters of a coding-related word, and
    ten "lives" (aka nine chances) to guess
    an incorrect letter before losing the
    game. The hardest part for me was
    JavaScript, since I had just begun
    learning it when I built this project.
    Making a hangman game turned out to
    be much more complicated than I had
    expected, but I'm glad I did it and I'm
    proud of how it turned out!
    Percent finished: 95%. Although it is
    functional, there are some details I
    would like to add, such as a stick figure
    "hangman" animation and restricting of
    input characters to letters.

  • Local Business

    Github | Website

    About this project

    Details: This is website for a local
    acupuncture practice. It was built with
    HTML and CSS, plus a little jQuery. Not
    having creative agency and figuring
    out how turn someone else's vision into
    a real website were new types of
    challenges. Although code-wise this was
    not my most challenging or exciting
    project, learning about the other types
    of obstacles that come up in this line
    of work was an invaluable experience.
    Percent finished: 90%. This website
    accomplishes everything the bussiness'
    owners and I agreed on beforehand.
    However, it's not responsive on all
    devices, which is very important for a
    deployed webiste. So far it's reponsive
    on iPad, iPad Pro, and iPhone X.

  • Fitness Friends

    GitHub | Application

    About this project

    Details: This is Fitness Friends, a
    social application for fitness lovers.

    portion of the work in React, and all of
    the styling. My favorite part of this
    project was working in React.
    Percent finished: 50%. We've
    laid most of the groundwork, but we
    have a ways to go before this
    application is finished. That being said,
    I'm still proud of the work I did,
    particulalry in React and Redux.

  • DoggyDate

    GitHub | Application
